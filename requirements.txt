# MediChat AI - Setup Instructions

## Prerequisites
- Node.js (v16.0.0 or higher)
- npm (v8.0.0 or higher) or Yarn (v1.22.0 or higher)
- Git

## Setup Instructions

### 1. Clone the repository
git+https://github.com/username/repo.git
cd SIH

### 2. Install Backend Dependencies
Go inside your backend folder `sih-backend` and run:

npm install express cors dotenv axios
npm install --save-dev nodemon

### 3. Install Frontend Dependencies
Go inside your frontend folder `sih` and run:

npm install
npm install axios

### 4. Environment Variables
Create a `.env` file in `sih-backend`:

GEMINI_API_KEY=YOUR_NEW_KEY_HERE
PORT=5000

> Replace `YOUR_NEW_KEY_HERE` with your Gemini API key from Google Cloud Console.

### 5. Run the Backend Server
From the `sih-backend` folder:

npx nodemon server.js

✅ You should see:

✅ Server running on http://localhost:5000

### 6. Run the Frontend (React App)
From the `sih` folder:

npm run dev

✅ Vite will start the dev server, and you’ll see:

VITE v5.0.0  ready in XXXms
Local:   http://localhost:5173/

Open this URL in your browser to test the chatbot.

## Troubleshooting

1. Dependency installation fails:
   - Clear npm cache: npm cache clean --force
   - Delete node_modules and package-lock.json
   - Run npm install again

2. Backend server not starting:
   - Check port 5000 is free
   - Ensure .env file exists with valid API key

3. Frontend server not starting:
   - Check port 5173 is free
   - Run npm install in frontend folder

4. Chatbot not responding:
   - Make sure backend is running
   - Verify Gemini API key is valid and not exceeded quota
